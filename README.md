This will track statistics for "A Link To The Past" Randomizer.  
It uses the websocket provided by QUSB2NES or USB2SNES.  
The Timer will start once the game begins either standard or open modes.
Once the Triforce room is opened the timer will stop. Timer can be opened in a seperate window
